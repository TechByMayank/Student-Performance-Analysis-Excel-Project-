
# 📊 Student Performance Analysis (Excel Project)

## 📌 Objective

To analyze student performance based on study hours, marks, and other attributes using Microsoft Excel.

---

## 📂 Dataset

The dataset contains the following columns:

* Student Roll Number
* Student Name
* Branch
* Course
* Study Hours
* Marks

---

## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Basic Excel Functions (IF, AVERAGE, COUNT)

---

## 📊 Analysis Performed

* Calculated average marks and study hours
* Categorized students based on performance (High/Medium/Low)
* Identified pass/fail students
* Created pivot tables for:

  * Branch-wise average marks
  * Course-wise study hours
  * Student count by category

---

## 📈 Key Insights

* Students with higher study hours generally scored better
* Some students achieved high marks with fewer study hours (efficient learners)
* Performance varies across different branches and courses
* Majority of students fall in the medium performance category

---

## 📊 Dashboard

An interactive dashboard was created using:

* Bar charts (Branch vs Marks)
* Pie chart (Performance distribution)
* Line chart (Study hours vs Marks)

---

## 📎 Project Structure

* `data/` → Raw dataset
* `analysis/` → Processed data and calculations
* `dashboard/` → Visualizations

---

## 🚀 Conclusion

This project demonstrates basic data analysis skills using Excel, including data cleaning, transformation, visualization, and insight generation.
